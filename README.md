# MooveGit

Criar uma app capaz de listar as informações do github.

<a href="https://carlosocarvalho.github.io/mooven-knownledge"> Demo online</a>

**Requisitos do desafio**

Usar no desenvolvimento 

- [x] RxJS
- [x] Pipe
- [x] Guards
- [x] Routes


Desenvolver páginas

- [x] pagina de pesquisa __(pesquisa sera um campo de autocomplete, ao selecionar o usuario. direcionar para a rota de lista de repositorios.)__
- [x] lista de repósitorios __(exibir uma para listar todos os itens do usuario)__
- [x] essa pagina deve ser de livre uso.
